While Loops

So far, your entire code runs only once, from top to bottom. This lesson will be on while loops, one of the most simple loops in programming. In a while loop, there is no condition that needs to be met to run. Instead, the code in the while loop will continuously loop until the condition is no longer met.

More Details

Within a while loop, you can also exit the loop with the break function. A break immediately "breaks" out of the loop, as the name suggests. This is usually used along with an if statement.
